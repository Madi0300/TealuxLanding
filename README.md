# Tealuxe Landing Page

Welcome to the **Tealuxe** landing page project! This is a responsive, modern website designed for a premium tea store, showcasing various tea collections, blog posts, and customer reviews.

## ðŸ“– About The Project

> **Note**: This is an educational pet project created for learning purposes.

Tealuxe is a concept website for a high-quality tea shop. The landing page is designed to provide a calming and elegant user experience, reflecting the nature of the products sold. It includes sections for browsing tea categories, reading about tea culture, and subscribing to newsletters.

### Key Features

*   **Responsive Design**: Fully adaptable layout that looks great on desktop, tablet, and mobile devices.
*   **Interactive Navigation**: Includes a mobile-friendly burger menu.
*   **Product Collections**: Visual showcase of different tea types (Black, Green, White, Oolong, Pu-erh).
*   **Blog Section**: Previews of the latest articles from the tea blog.
*   **Testimonials**: A section highlighting customer reviews and feedback.
*   **Newsletter Signup**: A functional-looking form for users to subscribe to updates.
*   **Information Sections**: Details about the company's priorities (Transparency, Quality, Support, Shipping).

## ðŸ›  Technologies Used

*   **CSS3**: Custom styling using the BEM (Block Element Modifier) methodology for maintainable code.
*   **JavaScript**: Vanilla JS for interactive elements like the mobile navigation menu.
